# cursor-setup-guide
Steps to setup Cursor IDE with Claude Code and Codex.

## Tools Installed
- Cursor IDE
- Claude Code extension (installed, not signed in)
- Codex extension (installed, not signed in)

## Steps Completed
1. Installed Cursor IDE
2. Enabled Extensions Marketplace
3. Added Claude Code extension
4. Added Codex extension
5. Verified Codex extension details (latest version installed)
6. Installed Git on Windows
7. Created public GitHub repository
8. Cloned repository into Cursor IDE
9. Opened repository folder in Cursor

## Issues & Solutions
- Claude Code requires a paid subscription → installed but not signed in
- Codex requires ChatGPT Plus or higher → installed but not signed in
- “Sign in with your ChatGPT account” text not clickable → documented as informational only
- Git not recognized initially → installed Git for Windows and added to PATH
